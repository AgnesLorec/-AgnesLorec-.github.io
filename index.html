Lac d'Estaing

Le lac d'Estaing est un lac de montagne situé dans les Pyrénées.

Informations

Localisation

Il est situé dans la vallée d'Estaing, dans le Val d'Azun.

Altitude

Le lac d'Estaing se situe à 1 161 m d'altitude.

Randonnée

Il est possible de faire le tour du Lac en 1h de marche. Le dénivelé est nul. La marche y est très facile.
